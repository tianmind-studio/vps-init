4 种内置 profile

看哪个场景贴就跑哪个。任何一个都能 --dry-run 先预览。

minimal

最小化 bootstrap。不动防火墙、不改 SSH。

  • Asia/Shanghai 时区
  • 阿里云镜像
  • 基础工具（不装 nginx）
  • 2G swap + swappiness=10

web-cn

最常用。完整 web VPS，配 site-bootstrap 刚好。

  • minimal 的全部
  • nginx + certbot
  • UFW + fail2ban（增强 bantime）
  • SSH key-only 加固

node-app

web-cn 再叠一层 Node.js 工具链。

  • web-cn 的全部
  • nvm 装 Node LTS
  • pnpm + pm2 全局
  • 适合 Next.js / Express

docker-host

Docker 宿主机。不装 nginx，走容器路线。

  • Asia/Shanghai + 阿里云镜像
  • Docker CE + buildx + compose
  • UFW + fail2ban
  • SSH 加固

中国场景做了哪些特别调优

欧美开源项目不会告诉你的那些细节。

apt 源双格式 兼容

Ubuntu 24.04 deb822 ubuntu.sources + 旧版 sources.list 自动识别。

多镜像可选 速度

aliyun / 清华 / 中科大 / 网易 / 华为云。默认阿里云（延迟最稳）。

fail2ban 升级 真实

bantime.increment=true。CN/HK VPS 被扫量级比欧美高一个数量级，10 分钟 bantime 根本不够。

小内存 swap 玄学

vm.swappiness=10。1G VPS 用默认 60 会疯狂换页，体感明显。

Docker 走官方源 版本

不从 apt install docker.io 装（版本落后 + 缺 buildx / compose plugin）。

SSH 防锁死 安全

如果所有用户都没 authorized_keys，拒绝关闭密码登录。把"我把自己锁在外面"这个经典坑堵死。
